# 🛡️ Pro-Vision: Embedded Camera Image Classification System
### Real-Time Driver Monitoring & Edge-Vision Analytics

This project is an **Image Classification Service** intended for low-resource or edge-like scenarios. It uses Computer Vision and a deployment-ready API architecture to classify human states into **AWAKE**, **DROWSY**, or **CRITICAL_SLEEP**.

---

## 🚀 Getting Started (Local Setup)

To test this system on your own machine, follow these steps:

### 1. Prerequisites
Ensure you have **Python 3.9** or higher installed on your system.

### 2. Installation
Clone this repository or download the files, then install the required dependencies:
```bash
pip install streamlit opencv-python numpy pandas
```

3. Run the Application
Launch the Streamlit dashboard by running:

```bash
python -m streamlit run Smile_detection.py
```
The dashboard will open automatically in your default web browser (usually at http://localhost:8501).
